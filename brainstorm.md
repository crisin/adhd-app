# ADHD App — Brainstorm

## Core Concept
An app designed around how ADHD brains actually work: time blindness, hyperfocus, task paralysis, difficulty transitioning between activities.

---

## Problem Areas to Address

- **Time blindness** — people with ADHD often lose track of time completely
- **Task initiation** — starting a task feels overwhelming
- **Task switching** — trouble stopping one thing and starting another
- **Overwhelm** — long todo lists feel paralyzing
- **Hyperfocus traps** — getting stuck on one thing and missing other commitments
- **Forgetting** — even important things vanish from working memory
- **Reward circuits** — standard todo apps offer no dopamine hit for completing things

---

## Feature Ideas

### Timers
- [ ] Pomodoro-style work/break cycles, but customizable for ADHD attention spans (e.g., 10/5 instead of 25/5)
- [ ] "Time anchor" — visual countdown so time feels real/tangible (e.g., shrinking color bar)
- [ ] Transition warnings — alert N minutes before a timer ends so you can mentally prepare
- [ ] Hyperfocus alarm — "you've been at this for X minutes, time to check in"

### Reminders
- [ ] Recurring smart reminders with escalating urgency (gentle → firm → alarm)
- [ ] Location-based reminders (e.g., "when you leave home, take meds")
- [ ] "Body double" mode — virtual co-working session feel with ambient sounds
- [ ] Snooze that asks *why* you're snoozing (context, not just delay)

### Todos / Tasks
- [ ] One task at a time — show only the next task, hide the rest
- [ ] "Brain dump" mode — quick capture with zero friction, sort later
- [ ] Task size estimation helper (ADHD people often underestimate)
- [ ] Break down big tasks into micro-steps automatically (could use AI)
- [ ] "Good enough" checkbox — done doesn't have to mean perfect

### Motivation / Engagement
- [ ] Celebration animations/sounds on task completion (dopamine reward)
- [ ] Streaks and gentle stats (not shame-y, only positive framing)
- [ ] "Just start for 2 minutes" — low-commitment task entry
- [ ] Points/XP system tied to completing focus sessions

### UX / Design Principles
- [ ] Minimal visual clutter — ADHD brains get overwhelmed by busy UIs
- [ ] Large, readable fonts and high contrast options
- [ ] Dark mode by default
- [ ] Keyboard shortcuts for everything (reduces friction)
- [ ] No notifications that are purely informational — every ping demands action
